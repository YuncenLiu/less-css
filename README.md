### less

less 是一种动态样式语言，属于CSS预处理器的范围，它扩展了css 语言，它扩展了 css 语言，增加了变量，minx，函数等特性，使css更易维护和扩展

[less 中文官网](https://lesscss.cn/)

[bootstrap 中文 less 官网](https://www.bootcss.com/p/lesscss/)

#### less注释

1. `//` 这种注释会被编译到 css 文件
2. `/**/` 这种注释不会被编译到 css 文件
